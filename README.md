# BeautyPro - Салон красоты

Приложение для управления салоном красоты с графическим интерфейсом на PySide6 и REST API на FastAPI.

## Структура проекта

```
Salon/
├── server/          # Backend (FastAPI)
│   ├── main.py      # API сервер
│   ├── models.py    # SQLAlchemy модели
│   ├── schemas.py   # Pydantic схемы
│   ├── crud.py      # CRUD операции
│   └── database.py  # Настройка БД
├── client/          # Frontend (PySide6)
│   ├── main.py      # GUI приложение
│   └── api_client.py # API клиент
└── requirements.txt # Зависимости
```

## Установка

1. **Создайте виртуальное окружение:**
```bash
python -m venv venv
```

2. **Активируйте виртуальное окружение:**

   **macOS/Linux:**
   ```bash
   source venv/bin/activate
   ```

   **Windows:**
   ```bash
   venv\Scripts\activate
   ```

3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

## Запуск проекта

### Шаг 1: Запуск сервера

Откройте первый терминал и выполните:

```bash
# Активируйте виртуальное окружение (если еще не активировано)
source venv/bin/activate  # macOS/Linux
# или
venv\Scripts\activate     # Windows

# Запустите сервер
python -m uvicorn server.main:app --reload --host 127.0.0.1 --port 8000
```

Сервер будет доступен по адресу: `http://127.0.0.1:8000`

API документация: `http://127.0.0.1:8000/docs`

### Шаг 2: Запуск клиентского приложения

Откройте второй терминал и выполните:

```bash
# Активируйте виртуальное окружение (если еще не активировано)
source venv/bin/activate  # macOS/Linux
# или
venv\Scripts\activate     # Windows

# Запустите клиент
python -m client.main
```

## Использование

1. **Регистрация/Вход:**
   - При первом запуске зарегистрируйтесь, указав номер телефона и пароль
   - Затем войдите в систему

2. **Запись на услугу:**
   - Перейдите в раздел "Новая запись"
   - Выберите мастера или услугу
   - Выберите дату и время
   - Подтвердите запись

3. **Просмотр истории:**
   - В разделе "История записей" можно просмотреть все свои записи
   - Можно отменить запланированные записи

4. **Администратор:**
   - Для доступа к админ-панели используйте специальный аккаунт администратора
   - В админ-панели можно управлять мастерами и услугами

## Технологии

- **Backend:** FastAPI, SQLAlchemy, SQLite
- **Frontend:** PySide6 (Qt for Python)
- **API:** REST API с автоматической документацией

## Примечания

- База данных SQLite создается автоматически при первом запуске сервера
- Сервер должен быть запущен перед запуском клиентского приложения
- Для Windows рекомендуется использовать стиль "Fusion" (уже настроен в коде)
